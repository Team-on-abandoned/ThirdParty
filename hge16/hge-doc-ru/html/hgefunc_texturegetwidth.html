HGE::Texture_GetWidth

Возвращает ширину текстуры в пикселях. 

int Texture_GetWidth(
  HTEXTURE texture,
  bool bOriginal = false
);

Параметры

texture
Идентификатор текстуры, у которой запрашивается ширина.
bOriginal
Если равно false или опущено, возвращается ширина текстуры в видеопамяти.
Если равно true, возвращается оригинальная ширина текстуры.

Возвращаемое значение

Возвращает ширину текстуры в пикселах.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
