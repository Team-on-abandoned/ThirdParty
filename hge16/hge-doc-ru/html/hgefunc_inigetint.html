HGE::Ini_GetInt

Читает целое значение из файла инициализации. 

int Ini_GetInt(
  const char *section,
  const char *name,
  int def_val
);

Параметры

section
Имя секции файла инициализации.
name
Имя ключа.
def_val
Значение по умолчанию.

Возвращаемое значение

Возвращает значение, прочитанное в файле инициализации. Если файл, секция или ключ не найдены, возварщается значение по умолчанию.

Примечания

Переменная системы HGE_INIFILE должна быть устрановлена перед вызовом этой функции. Имя секции регистро-независимое.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
