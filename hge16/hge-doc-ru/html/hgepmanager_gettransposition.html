hgeParticleManager::GetTransposition

Возвращает текущий сдвиг координат. 

void GetTransposition(
  float *dx,
  float *dy
);

Параметры

dx
Указатель на переменную типа float, для сохранения значения перемещения X.
dy
Указатель на переменную типа float, для сохранения значения перемещения Y.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib
