Класс hgeGUIText

hgeGUIText является классом элемента текста GUI. Чтобы создать объект класса hgeGUIText можно воспользоватья конструктором: 

hgeGUIText(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  hgeFont*  fnt
);

Параметры

id
Целый идентификатор для созданного элемента управления.
x
Координата x левого верхнего угла элемента управления.
y
Координата y левого верхнего угла элемента управления.
w
Ширина элемента управления.
h
Высота элемента управления.
fnt
Указатель на объект класса hgeFont, который будет использован для рисования текста.

Другие методы

SetMode Изменяет режим выравнивания текста.
SetText Изменяет текст элмента.
printf Печатает форматированную строку в текст элемента.
SetColor Изменяет цвет элемента текста.

Требования

Заголовок: hgeguictrls.h
Библиотека импорта: hgehelp.lib
