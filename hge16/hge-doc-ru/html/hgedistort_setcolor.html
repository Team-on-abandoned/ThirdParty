hgeDistortionMesh::SetColor

Устанавливает цвет и альфа указанному узлу сетки искажения. 

void SetColor(
  int col,
  int row,
  DWORD color
);

Параметры

col
Номер столбца требуемого узла сетки. Нумерация начинается с 0.
row
Номер строки требуемого узла сетки. Нумерация начинается с 0.
color
Новое значение цвета.

Примечания

Цвет включает и альфу. Смотрите раздел Hardware color format для более подробной информации. Способ, который применяется для рисования текстуры зависит от режима смешивания. Смотрите секцию Режимы смешивания для более подробной информации.

Библиотека импорта: hgehelp.lib
