HGE::System_Launch

Открывает URL или стартует выполняемый/дата файл. 

void System_Launch(
  const char *url
);

Параметры

url
Имя выполняемого файла, файла с данными или интернет адреса URL, который необходимо открыть при помощи назначенного приложения по умолчанию.

Примечания

Можно использовать функцию System_Launch для открытия Web сайтов в браузере, назначенном по умолчанию, или запуска почтового клиента с новым сообщением. Просто используйте стандартный синтаксис URL.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
