hgeDistortionMesh::SetDisplacement

Устанавливает перемещение для указанного узла сетки искажения. 

void SetDisplacement(
  int col,
  int row,
  float dx,
  float dy,
  int ref
);

Параметры

col
Номер столбца узла сетки, который надо изменить. Нумерация начинается с 0.
row
Номер строки узла сетки, который надо изменить. Нумерация начинается с 0.
dx
Перемещение вдоль оси X.
dy
Перемещение вдоль оси Y.
ref
Относительная точка перемещения. Принимает одно из значений:
HGEDISP_NODE - относительная точка является исходной позицией узла
HGEDISP_TOPLEFT - относительная точка является левым верхним узлом сетки
HGEDISP_CENTER - относительная точка является центром сетки искажения

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib
