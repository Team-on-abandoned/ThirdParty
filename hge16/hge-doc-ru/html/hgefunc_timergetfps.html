HGE::Timer_GetFPS

Возвращает значение текущего FPS (кадров-в-секунду). 

int Timer_GetFPS();

Возвращаемое значение

Возвращает количество вызовов функции кадров, за последнюю секунду.

Примечания

Используйте Timer_GetFPS внутри функции кадра для вывода информации и отладки. Разрешение таймера - 1 милисекунда, поэтому значение не может привысить 1000 FPS.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
