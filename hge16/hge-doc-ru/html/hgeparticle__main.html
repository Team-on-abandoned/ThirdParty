Вспомогательный класс hgeParticleSystem

hgeParticleSystem является вспомогательным классом HGE для продвинутых 2Д систем частиц.

Методы класса hgeParticleSystem

Конструкторы Создают и инициализируют объекты класса hgeParticleSystem.
Операторы Операторы класса hgeParticleSystem.
  
Update Обновляет систему частиц.
MoveTo Перемещает систему частиц в новое положение.
Transpose Добавляет к координатам смещение.
FireAt Включает систмеу частиц в некоторой позиции на экране.
Fire Запускает систему частиц.
Stop Останавливает систему частиц.
Render Рисует систему частиц на экране.
TrackBoundingBox Включает/выключает отслеживание ограничивающего прямоугольника для системы частиц.
  
GetBoundingBox Возвращает текущий ограничивающий прямоугольник системы частиц.
GetParticlesAlive Возвращает число активных частиц системы.
GetPosition Возвращает текущую позицию системы частиц.
GetTransposition Возвращает текущий сдвиг системы частиц.
GetAge Возвращает возраст системы частиц.
