hgeSprite::Render

Рисует спрайт на экране. 

void Render(
  float x,
  float y
);

Параметры

x
Экранная координата x, где нужно нарисовать спрайт.
y
Экранная координата y, где нужно нарисовать спрайт.

Примечания

Метод Render автоматически сдвигает координаты на -0.5, чтобы добиться точного отображения текселей в пикселы, не затрагивая тексуру с интерполяцией.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib

Смотрите также

RenderEx, RenderStretch, Render4V
