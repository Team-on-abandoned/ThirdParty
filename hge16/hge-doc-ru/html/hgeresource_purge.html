hgeResourceManager::Purge

Удаляет все ресурсы указанной группы из памяти. 

void Purge(
  int groupid = 0
);

Параметры

groupid
Идентификатор группы ресурсов, которые необходимо удалить из памяти. Если равен 0 или опущен, все ранее созданные ресурсы будут удалены из памяти.

Требования

Заголовок: hgeresource.h
Библиотека импорта: hgehelp.lib
