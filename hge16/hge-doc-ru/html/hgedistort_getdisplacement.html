hgeDistortionMesh::GetDisplacement

Возвращает перемещение указанного узла сетки искажения. 

void GetDisplacement(
  int col,
  int row,
  float *dx,
  float *dy,
  int ref
);

Параметры

col
Номер столбца узла сетки. Нумерация начинается с 0.
row
Номер строки узла сетки. Нумерация начинается с 0.
dx
Указатель на переменную типа float куда должно быть сохранено перемещение вдоль оси X.
dy
Указатель на переменную типа float куда должно быть сохранено перемещение вдоль оси Y.
ref
Точка относительного движения. Принимает одно из следующих значений:
HGEDISP_NODE - относительная точка является позицией узла по умолчанию
HGEDISP_TOPLEFT - относительная точка является левым верхним углом сетки
HGEDISP_CENTER - относительная точка является центром сетки

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib

SetDisplacement, GetZ, GetColor
