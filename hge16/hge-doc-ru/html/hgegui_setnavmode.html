hgeGUI::SetNavMode

Устанавливает режим навигации в GUI. 

void SetNavMode(
  int navmode
);

Параметры

navmode
Побитовая (|) комбинация следующих констант:
HGEGUI_NONAVKEYS - без навигации с клавиатуры
HGEGUI_LEFTRIGHT - стрелки влево и вправо переключают пердвигают курсор по элементам управления
HGEGUI_UPDOWN - стрелки вверх и вниз переключают пердвигают курсор по элементам управления
HGEGUI_CYCLED - перемещение курсора круговое

Примечания

По умолчанию задан режим HGEGUI_NONAVKEYS.

Требования

Заголовок: hgegui.h
Библиотека импорта: hgehelp.lib
