hgeParticleSystem::TrackBoundingBox

Включает/выключает отслеживание ограничивающего прямоугольника. 

void TrackBoundingBox(
  bool bTrack
);

Параметры

bTrack
Если равно true, отслеживание ограничивающего прямоугольника будет включено.
Если равно false, отслеживание ограничивающего прямоугольника будет выключено.

Примечания

В целях повышения производительности, отслеживание ограничивающего прямоугольника выключено по умолчанию. Если вам надо использовать эту функциональность, необходимо включить ее явным образом.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib
