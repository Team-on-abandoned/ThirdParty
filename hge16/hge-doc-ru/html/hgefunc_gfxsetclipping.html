HGE::Gfx_SetClipping

Устанавливает область отчечения для всех графических объектов, выводимых далее. 

void Gfx_SetClipping(
  int x = 0,
  int y = 0,
  int w = 0,
  int h = 0
);

Параметры

x
X координата левой верхней точки области отсечения.
y
Y координата левой верхней точки области отсечения.
w
Ширина области отсечения.
h
Высота области отсечения.

Примечания

Если все параметры равны 0 или опущены, область отсечения сбрасывается до размеров всей цели рендеринга. Когда цель рендеринга переключается, область отсечения так же сбрасывается до всей цели рендеринга.

Gfx_SetClipping сбрасывает в видеоадаптер все накопленные графические примитивы.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
