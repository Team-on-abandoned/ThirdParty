hgeGUI::AddCtrl

Adds a control to the GUI. 

void AddCtrl(
  hgeGUIObject *ctrl
);

Параметры

ctrl
Pointer to a GUI control object to be added to the GUI.

Примечания

The control is an object of any class derived from hgeGUIObject class. All added controls are removed from memory within the GUI destructor.

Требования

Заголовок: hgegui.h
Библиотека импорта: hgehelp.lib
