Сборка пакета ресурсов

Пакеты ресурсов HGE - простые ZIP-архивы. Так что можете пользоваться для создания и управления любым ZIP архиватором, который вам нравится. Чтобы создать пакет используя командную строчку архиватора pkzip, можете использовать следующий синтаксис: 

pkzip -r -p pack.paq Data\*.*
-r
Поддиректории ресурсов
-p
Сохранять пути (исключая путь базовой директории)
pack.paq
Название выходного файла
Data\*.*
Маска добавляемых файлов

Учтите, что можно пользоваться криптованием с помошью пароля, хотя загрузка таких ресурсов будет немного медленнее обычных.
