hgeSprite::GetBoundingBox

Возвращает ограничивающий прямоугольник спрайта. 

hgeRect *GetBoundingBox(
  float x,
  float y,
  hgeRect *rect
);

Параметры

x
Экранная координата x спрайта.
y
Экранная координата y спрайта.
rect
Указатель на объект hgeRect куда будет записан ограничивающий прямоугольник.

Возвращаемое значение

Возврашает параметр rect.

Примечания

Ограничивающий прямоугольник вычисляется принимая во внимание центр спрайта. Смотрите медтод подробнее SetHotSpot.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib
