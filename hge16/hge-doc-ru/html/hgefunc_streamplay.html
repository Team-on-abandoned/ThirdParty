HGE::Stream_Play

Начинает воспроизводить аудио поток. 

HCHANNEL Stream_Play(
  HSTREAM stream,
  bool loop
);

Параметры

stream
Идентификатор аудио потока, который надо воспроизводить.
loop
Если равно true, поток воспроизводится циклически, до тех пор, пока не будет остановлен вручную
Если равно false поток воспроизводится один раз.

Возвращаемое значение

Возвращает идентификатор аудио канала, через которы воспроизводится поток. Используйте это значение в функциях управления аудио каналами.

Примечания

Только один экземпляр потока может быть воспроизведен одновременно Если поток уже воспроизводится, он перезапустится с параметрами по умолчанию.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
