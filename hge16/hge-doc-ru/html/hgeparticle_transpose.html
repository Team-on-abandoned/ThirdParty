hgeParticleSystem::Transpose

Добавляет постоянный сдвиг в координаты системы частиц. Эта функция может оказаться полезной в играх с прокручивающейся картой/уровнем. 

void Transpose(
  float dx,
  float dy
);

Параметры

dx
Координата перемещения X.
dy
Координата перемещения Y.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib

Смотрите также

GetTransposition, MoveTo, GetPosition
