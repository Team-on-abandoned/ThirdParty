hgeRect::Intersect

Проверяет два ограничивающих прямоугольника на пересечение. 

bool Intersect(
  hgeRect *rect
);

Параметры

rect
Указатель на ограничивающий прямоугольник, с которым проверяется пересечение.

Возвращаемое значение

Если ограничивающие прямоугольники пересекаются, возвращается true. В противном случае возвращается false.

Требования

Заголовок: hgerect.h
Библиотека импорта: hgehelp.lib
