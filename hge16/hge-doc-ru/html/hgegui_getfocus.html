hgeGUI::GetFocus

Возвращает текущий фокус ввода. 

int GetFocus();

Возвращаемое значение

Идентификатор элемента управления, который владеет фокусом ввода.

Примечания

События клавиатуры обрабатываются только в элементе управления, который владеет фокусом.

Требования

Заголовок: hgegui.h
Библиотека импорта: hgehelp.lib
