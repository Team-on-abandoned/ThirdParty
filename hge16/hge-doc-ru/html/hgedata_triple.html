Структура треугольника в HGE

Треугольник в HGE - произвольно окрашенный и текстурированный 2Д треугольник. Для описания треугольника используется структура hgeTriple: 

struct hgeTriple
{
  hgeVertex  v[3];
  HTEXTURE   tex;
  int        blend;
};

Члены

v
Массив вершин, описывающих треугольник.
tex
Идентификатор текстуры треугольника или 0.
blend
Режим смешивания треугольника.

Требования

Заголовок: hge.h
