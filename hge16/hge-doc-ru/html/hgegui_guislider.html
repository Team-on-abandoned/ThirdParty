Класс hgeGUISlider

hgeGUISlider является классом элмента бегунка GUI. Чтобы создать объект класса hgeGUISlider можно использовать конструктор: 

hgeGUISlider(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  HTEXTURE  tex,
  float     tx,
  float     ty,
  float     sw,
  float     sh
);

Параметры

id
Числовой идентификатор созданного элемента управления.
x
Координата x левого верхнего угла бегунка.
y
Координата y левого верхнего угла бегунка.
w
Ширина бегунка.
h
Высота бегунка.
tex
Идентификатор текстуры, содержащей изображение бегунка. В зависимости от режима бегунка, изображение может интерпретироваться по-разному. Смотриет SetMode для более детального описания.
tx
Текстурная координата x левого верхнего угола изображения бегунка.
ty
Текстурная координата y левого верхнего угола изображения бегунка.
sw
Ширина изображения бегунка.
sh
Высота изображения бегунка.

Другие методы

SetMode Меняет режим поведения бегунка.
SetValue Меняет позицию бегунка.
GetValue Возвращает текущую позицию бегунка.

Требования

Заголовок: hgeguictrls.h
Библиотека импорта: hgehelp.lib
