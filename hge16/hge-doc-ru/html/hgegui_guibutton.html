Класс hgeGUIButton

hgeGUIButton является классом кнопки GUI. Чтобы создать объект класса hgeGUIButton можно воспольщоваться конструктором: 

hgeGUIButton(
  int       id,
  float     x,
  float     y,
  float     w,
  float     h,
  HTEXTURE  tex,
  float     tx,
  float     ty
);

Параметры

id
Идентификатор созданного элемента управления.
x
X координата левого верхнего угла кнопки.
y
Y координата левого верхнего угла кнопки.
w
Ширина кнопки.
h
Высота кнопки.
tex
Идентификатор текстуры, содержащий изображение кнопки. Текстура должна содержать два изображения, одно для кнопки в ненажатом состоянии и точно справа от него изображение для кнопки в нажатом состоянии.
tx
Координата x левого верхнего угла изображения кнопки в пикселях.
ty
Координата y левого верхнего угла изображения кнопки в пикселях.

Другие методы

SetMode Меняет режим поведения кнопки.
SetState Меняет состояние кнопки.
GetState Возвращает состояние кнопки.

Требования

Заголовок: hgeguictrls.h
Библиотека импорта: hgehelp.lib
