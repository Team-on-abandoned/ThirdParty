HGE::System_SetState

Устанавливает внутреннюю переменную системы. 

void System_SetState(
  hgeState state,
  hgeValue value
);

Параметры

state
Значение перечисления hgeState. Определяет необходимую переменную системы.
value
hgeValue - объект, содержащий необходимое значение переменной.

Примечания

Используйте данную функцию для установки необходимого значения внутренних переменных системы, таких как разрешение экрана, заголовок окна, название файла протокола и т.д. И хотя многие из них имеют значения по умолчанию, как мимнимум переменная HGE_FRAMEFUNC должна быть установлена перед вызовом System_Start.

Смотри секцию System states полного описания переменных системы.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
