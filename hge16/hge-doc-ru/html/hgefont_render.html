hgeFont::Render

Рисует строку текста на экране. 

void Render(
  float x,
  float y,
  const char *string,
  int align = HGETEXT_LEFT
);

Параметры

x
Экранная координата x.
y
Экранная координата y.
string
Страка для отображения.
align
Необязательный параметр. Способ выравнивания Может принимать значения: HGETEXT_LEFT, HGETEXT_RIGHT, HGETEXT_CENTER. Значение по умолчанию HGETEXT_LEFT.

Примечания

Разделитель строк ('\n') корректно обрабатывается. Все другие символы управления строкой рассматриваются как обычные символы. Неопределенные символы будут отображены как '?'.

Требования

Заголовок: hgefont.h
Библиотека импорта: hgehelp.lib
