HGE::Music_Play

Начинает воспроизведение музыки. 

HCHANNEL Music_Play(
  HMUSIC music,
  bool loop
);

Параметры

music
Идентификатор музыки, которую нужно воспроизводить.
loop
Если значение равно true, музыка будет воспроизводиться циклически, до тех пор, пока канал не будет остановлен вручную.
Если false музыка будет воспроизведена один раз.

Возвращаемое значение

Возвращает идентификатор аудио канала, через который воспроизводится музыка. Используйте идентификатор в функциях управления аудио каналами.

Примечания

Только один музыкальный экземпляр может воспроизводиться одновременно. Если музыка уже играет, ее можно начать воспроизводить заново, используя параметры по умолчанию.

Требования

Библиотека импорта: hge.lib
