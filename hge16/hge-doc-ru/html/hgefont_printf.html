hgeFont::printf

Рисует строку форматированного текста. 

void printf(
  float x,
  float y,
  const char *format,
  ...
);

Параметры

x
Экранная координата x.
y
Экранная координата y.
format
Форматированная в стиле функции printf языка C строка.
...
Необязательный список аргументов форматной строки.

Примечания

Смотрите справку по компилятору или руководство по языку С для более детальной информации о спецификации форматирования.

Требования

Заголовок: hgefont.h
Библиотека импорта: hgehelp.lib
