Конструкторы класса hgeSprite

Чтобы создать и проинициализировать объект класса hgeSprite можно использовать один из следующих синтаксисов: 

hgeSprite(
  HTEXTURE tex,
  float x,
  float y,
  float w,
  float h
);

hgeSprite(
  const hgeSprite &sprite
);

Параметры

tex
Идентификатор текстуры содержащей спрайт или 0.
x
Текстурная координата x в текселях.
y
Текстурная координата y в текселях.
w
Ширина спрайта.
h
Высота спрайта.

Примечания

Если tex = 0, то в качестве текстуры используется белый цвет.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib
