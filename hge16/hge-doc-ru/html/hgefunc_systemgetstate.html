HGE::System_GetState

Возвращает значение внутренней переменной системы. 

hgeValue System_GetState(
  hgeState state
);

Параметры

state
Член перечисления hgeState. Определяет требуемую переменную системы.

Возвращаемое значение

Возварщает объект класса hgeValue, содержащий значение требуемой перменной.

Примечания

Смотри секцию System states полного описания переменных системы.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib

System_SetState, System states, System state value class
