hgeSprite::GetColor

Возвращает цвет требуемой вершины спрайта. 

DWORD GetColor(
  int i = 0
);

Параметры

i
Необязательный индекс вершины спрайта. Вершины проиндексированы по часовой стрелки, начиная с левого верхнего угла и имеют значения 0-3. Если параметр опущен, то возвращается цвет первой вершины.

Примечания

Возвращает цвет указанной вершины спрайта.

Примечания

Цветовой параметр включает значение альфы. Смотрите секцию Аппаратный формат цвета для более подробной информации. Способ, который применяется для раскаршивания зависит от режима смешивания спрайта. Смотрите секцию Режимы смешивания для более подробной информации.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib
