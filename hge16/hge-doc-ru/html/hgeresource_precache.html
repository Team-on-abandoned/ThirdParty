hgeResourceManager::Precache

Создает все ресурсы указанной группы в памяти. 

bool Precache(
  int groupid = 0
);

Параметры

groupid
Идентификатор группы ресурсов, которую надо создать в памяти. Если номер равен 0 или опущен, то все ресурсы объявленные в скрипте будут созданы.

Возвращаемое значение

Если все ресурсы были успешно загружены, вернется true. В противном случае false.

Требования

Заголовок: hgeresource.h
Библиотека импорта: hgehelp.lib
