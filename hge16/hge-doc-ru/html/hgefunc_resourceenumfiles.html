HGE::Resource_EnumFiles

Перечисляет файлы по заданной маске. 

char *Resource_EnumFiles(
  const char *wildcard = 0
);

Параметры

wildcard
Абсолютная или относительная маска, включающаяя специальные символы "?" and "*" для начала нового перечисления или 0, чтобы продолжить начатое перечисление.

Возвращаемое значение

Возвращает следующее найденное имя файла или 0, если больше нет файлов.

Примечания

Resource_EnumFiles возвращает только имена файлов без их пути. Оно также пропускает директории, используйте Resource_EnumFolders для перечисления директорий.

Для перечисления всех файлов в некоторй директории, используйте символ "*" в маске. Пример: Resource_EnumFiles("data\*") начнет перечисление всех файлов в директории "data" которая расположена в директории приложения.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
