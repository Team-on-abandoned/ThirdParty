Visual C++ 6.0

1. Установка путей к библиотекам и закголовкам

Откройте закладку Tools->Options->Directories, и добавьте путь до HGE директории "include" и передвиньте ее наверх:


На этой же вкладке выберите Library files из выподающего списка Show directories for , добавьте путь к HGE директории "lib\vc" и передвиньте его наверх:


Нажмите OK.

Теперь можно использовать HGE в ваших проектах без определения его точного местоположения. Этот шаг нужно сделать только один раз для всех проектов.

2. Создание пустого проекта Win32 Application

Откройте закладку File->New...->Projects, выберите Win32 Application, укажите Project name и Location:


Нажмите OK.

На следующем экране выберите An empty project или A simple Win32 application если вы хотите, чтобы была просто создана структура:


Нажмите Finish. Вуаля! У вас создан новый проект.

3. Добавление библиотек в проект

Откройте закладку Project->Settings...->Link и добавьте библиотеки в Object/library modules:

  • hge.lib - библиотека импорта для DLL системного слоя HGE
  • hgehelp.lib - библиотека вспомогательных классов HGE

Если вы не вызываете никакие функции Windows, то эти две библиотеки - все что нужно для проекта.


Затем выбериет Win32 Release из выподающего списка Settings For и повторите операции.

Нажмите OK.

4. Начинайте писать код!

Включите заголовок hge.h для доступа к функциям HGE. Смотрите секцию Примеры простыми примерами кода.
