hgeFont::GetSprite

Возвращает объект hgeSprite для указанного символа. 

hgeSprite *GetSprite(
  char chr
);

Параметры

chr
Символ, для которого запрашивается спрайт.

Возвращаемое значение

Возвращается указатель на объект hgeSprite для заданного символа.

Требования

Заголовок: hgefont.h
Библиотека импорта: hgehelp.lib
