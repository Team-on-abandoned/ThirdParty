hgeDistortionMesh::GetColor

Возвращает цвет указанного узла. 

DWORD GetColor(
  int col,
  int row
);

Параметры

col
Номер столбца узла. Нумерация начинается с 0.
row
Номер строки узла. Нумерация начинается с 0.

Возвращаемое значение

Возваращает значение цвета для указанного узла сетки.

Примечания

Цвет включает альфа. Смотрите секцию Аппаратный формат цвета для более детальной информации.

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib

Смотрите также

SetColor, GetZ, GetDisplacement, Аппаратный формат цвета
