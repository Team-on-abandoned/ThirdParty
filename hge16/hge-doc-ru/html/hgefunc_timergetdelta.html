HGE::Timer_GetDelta

Возвращает время, прошедшее с последнего вызова функции кадра. 

float Timer_GetDelta();

Возвращаемое значение

Возвращает время, прошедшее с последнего вызова функуии кадра в секундах.

Примечания

Используйте Timer_GetDelta внутри функции кадра для синхронизации с различной скоростью выполнения и определения периодов времени. Разрешение таймера - 1 милисекунда, поэтому это значение не может быть меньше 0.001.

Библиотека импорта: hge.lib
