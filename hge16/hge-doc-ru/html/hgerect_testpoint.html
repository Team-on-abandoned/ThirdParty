hgeRect::TestPoint

Проверяет, лежит ли точка внутри ограничивающего прямоугольника. 

bool TestPoint(
  float x,
  float y
);

Параметры

x
Координата X точки.
y
Координата Y точки.

Возвращаемое значение

Если точка лежит внутри ограничивающего прямоугольника, то возвращается true. В противном случае возвращается false.

Требования

Заголовок: hgerect.h
Библиотека импорта: hgehelp.lib
