Вспомогательный класс hgeResourceManager

hgeresourceManager является вспомогательным классом HGE для автоматизации создания сложных ресурсов и управления ими в памяти.

Методы hgeResourceManager

Конструктор Создает и инициализирует объекты класса hgeResourceManager.
  
ChangeScript Меняет текущий скрипт ресурсов.
Precache Создает все ресурсы указанной группы в памяти.
Purge Удаляет все ресурсы указанной группы в памяти.
  
GetResource Возвращает указатель на ресурс.
GetTexture Возвращает идентификатор текстуры по имени.
GetEffect Возвращает идентификатор звукового эффекта по имени.
GetMusic Возвращает идентификатор музыки по имени.
GetStream Возвращает идентификатор звукового потока по имени.
GetTarget Возвращает идентификатор цели рендеринга по имени.
GetSprite Возвращает указатель на спрайт по имени.
GetAnimation Возвращает указатель на анимацию по имени.
GetFont Возвращает указатель на шрифт по названию.
GetParticleSystem Возвращает указатель на систему частиц по имени.
GetDistortionMesh Возвращает указатель на сетку искажения по имени.
