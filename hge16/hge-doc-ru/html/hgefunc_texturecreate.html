HGE::Texture_Create

Создает пустую текстуру. 

HTEXTURE Texture_Create(
  int width,
  int height
);

Параметры

width
Ширина создаваемой текстуры.
height
Высота создаваемой текстуры.

Возвращаемое значение

В случе успеха, возвращается идентификатор текстуры. В противном случае 0.

Примечания

Значения размеров текстуры должно быть степенью 2. Если это условие не выполнено, текстура втоматически дополняется до ближайшего значени степени 2, большего заданного.

Можно использовать функцию Texture_Lock и Texture_Unlock чтобы заполнить данными пустую текстуру.

Если текстура больше не нужна, необходимо удалить ее вызвав функцию Texture_Free.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib

Смотрите также

Texture_Load, Texture_Free, Texture_Lock, Texture_Unlock
