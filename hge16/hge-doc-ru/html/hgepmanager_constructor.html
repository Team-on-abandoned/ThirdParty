Конструктор класса hgeParticleManager

Чтобы создать и проинициализировать объект класса hgeParticleManager можно воспользоваться конструктором: 

hgeParticleManager(
  float fps = 0
);

Parameters

fps
Частота обновления систем частиц (количество кадров в секунду). Используйте этот параметр, чтобы сделать работу систем частиц независимой от частоты кадров и уменьшить вычисления на немного. Если параметр равен 0 или опущен, системы частиц обновляются каждый раз при вызове Update.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib
