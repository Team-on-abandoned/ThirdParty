hgeDistortionMesh::SetZ

Устанавливает Z-глубину указанного узла сетки. 

void SetZ(
  int col,
  int row,
  float z
);

Параметры

col
Номер столбца узла сетки. Нумерация начинается с 0.
row
Номер столбца узла сетки. Нумерация начинается с 0.
z
Z-глубина, в пределах от 0.0 до 1.0.

Примечания

Если z-буфер не используется, значение z-глубины игнорируется.

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib

Смотрите также

GetZ, SetColor, SetDisplacement
