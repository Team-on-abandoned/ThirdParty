Конструкторы класса hgeAnimation

Чтобы создать и проинициализировать объект класса hgeAnimation можно воспользоваться одним из следующих конструкторов: 

hgeAnimation(
  HTEXTURE tex,
  int nframes,
  float FPS,
  float x,
  float y,
  float w,
  float h
);

hgeAnimation(
  const hgeAnimation &anim
);

Параметры

tex
Идентификатор текстуры содержащей кадры анимации.
nframes
Количество кадров анимации.
FPS
Скорость анимации (количество кадров в секунду).
x
Текстурная координата x первого кадра анимации в текселях.
y
Текстурная координата y первого кадра анимации в текселях.
w
Длина анимационного кадра.
h
Высота анимационного кадра.

Примечания

Предполагается, что кадры анимации расположены в текстуре последовательно, слева на право и сверху вниз. Если для следующего кадра справа нету места, то кадр берется в следующей строке, начиная с позиции x=0.

Требования

Заголовок: hgeanim.h
Библиотека импорта: hgehelp.lib
