hgeSprite::RenderStretch

Рисует спрайт на экране с наклоном. 

void RenderStretch(
  float x1,
  float y1,
  float x2,
  float y2
);

Параметры

x1
Экранная координата x левого верхнего угла спрайта.
y1
Экранная координата y левого верхнего угла спрайта.
x2
Экранная координата x правого нижнего угла спрайта.
y2
Экранная координата y правого нижнего угла спрайта.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib

Смотрите также

Render, RenderEx, Render4V
