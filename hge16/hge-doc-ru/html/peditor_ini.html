Файлы инициализации Редактора Систем Частиц HGE

Файл инициализации

Редактор Систем Частиц HGE содержит параметры настройки в файле particleed.ini. Поэтому вы можете изменять их, настраивая тем самым поведение редактора. На данный момент, поддерживаются следующие параметры:

FullScreen
Если равно 1, редактор запускается в полноэкранном режиме. Если равно 0, редактор запускается в оконном режиме. Значение по умолчанию 0.
UpdatesPerSec
Определяет число обнавлений системы частиц в секунду. Если равно 0, число обновлений неограничего и зависит от скорости выполнения. Значение по умолчанию 50.

Файл с изображением частиц

Другой файл, который может вас заинтересовать - файл particles.png, который содержит спрайты. Он состоит из 16-ти изображений размеров 32x32, которые используются для отображения системы частиц в редакторе. Вы можете изменять любое из изображений или весь файл целиком. По умолчанию представлены моно-цветные изображения, но вы можете использовать и цветные. Запомните, при сохранении PNG, необходимо записать и альфаканал, если вы планируете использовать не только режим смешивания "добавление".
