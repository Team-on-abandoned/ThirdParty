hgeResourceManager::GetResource

Возвращает указатель на простой ресурс по имени. 

void *GetResource(
  const char *name
);

Параметры

name
Название простого ресурса.

Возвращаемое значение

В случае успеха, возвращает указатель на простой ресурс в памяти. В противном случае 0.

Примечания

Вы можете использовать имя объявленное в скрипте ресусов или просто имя файла, абсолютное или относительное к директории приложнеия или корню архива ресурсов.

Если ресурс еще не загружен, то он будет загружен. В противном случае вернется указатель на существующий ресурс.

Требования

Заголовок: hgeresource.h
Библиотека импорта: hgehelp.lib
