hgeResourceManager::GetParticleSystem

Возвращает указатель на систему частиц по имени. 

hgeParticleSystem *GetParticleSystem(
  const char *name
);

Параметры

name
Название системы частиц.

Возвращаемое значение

В случае успеха, возвращает указатель на объект класса hgeParticleSystem. В противном случае 0.

Примечания

Чтобы использовать систему частиц в hgeResourceManager, необходимо объявить ее в скрипте ресурсов.

Если система частиц еще не была создана, то она будет создана. В противном случае будет возвращен указатель на существующую сиетму частиц.

Можно использовать член класса info в возвращенном указателе, чтобы создать много экземпляров объектов класса hgeParticleManager.

Требования

Заголовок: hgeresource.h
Библиотека импорта: hgehelp.lib
