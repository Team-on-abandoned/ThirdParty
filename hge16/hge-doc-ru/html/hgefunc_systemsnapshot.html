HGE::System_Snapshot

Сохраняет снимок текущего экрана в файл. 

void System_Snapshot(
  const char *filename = 0
);

Параметры

filename
Абсолютный или относительный путь к файлу, куда необходимо сохранить снимок. Если равен 0 или опущено, снимок записывается в директорию приложения под именем shotXXX.bmp, где XXX следющее свободное число.

Примечания

Снимок записывается в файле формата BMP.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
