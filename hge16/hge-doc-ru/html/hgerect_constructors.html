Конструкторы класса hgeRect

Чтобы создать и проинициализирова объект класса hgeRect можете воспользоваться одним из следующих конструкторов: 

hgeRect();

hgeRect(
  float x1,
  float y1,
  float x2,
  float y2
);

hgeRect(
  const hgeRect &rect
);

Параметры

x1
Координата X левого верхнего угла.
y1
Координата Y левого верхнего угла.
x2
Координата X правого нижнего угла.
y2
Координата Y правого нижнего угла.

Требования

Заголовок: hgerect.h
