HGE::Timer_GetTime

Returns time elapsed since System_Initiate function call. 

float Timer_GetTime();

Возвращаемое значение

Возвращает время в секундах, прошедшее после выполнения функции System_Initiate.

Примечания

Пользуйтесь Timer_GetTime для определения периодов времени или времени, прошедшего с начала главного цикла игры. Разрешение таймера - 1 милисекунда.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
