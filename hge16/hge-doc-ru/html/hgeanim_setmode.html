hgeAnimation::SetMode

Устанавливает режим воспроизведения анимации. 

void SetMode(
  int mode
);

Параметры

mode
Побитная комбинация (|) констант:
HGEANIM_FWD - Воспроизводить в обычном направлении
HGEANIM_REV - Воспроизводить в обратном направлении
HGEANIM_PINGPONG - Цикл "туда-сюда"
HGEANIM_NOPINGPONG - Не зацикливать "туда-сюда"
HGEANIM_LOOP - Циклическя анимация
HGEANIM_NOLOOP - Не зацикливать анимацию

Примечания

Значение по умолчанию - HGEANIM_FWD | HGEANIM_LOOP.

Требования

Заголовок: hgeanim.h
Библиотека импорта: hgehelp.lib
