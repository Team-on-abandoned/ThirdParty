HGE::Ini_SetString

Записывает строку в файл инициализации. 

void Ini_SetString(
  const char *section,
  const char *name,
  const char *value
);

Параметры

section
Имя секции в файле инициализации.
name
Имя ключа.
value
Строка для записи.

Примечания

Переменная системы HGE_INIFILE должна быть устрановлена перед вызовом этой функции.

Если файл инициализации, секция или ключ не существуют, они создадутся. Имя секции регистро-независимое.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
