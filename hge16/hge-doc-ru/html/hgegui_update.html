hgeGUI::Update

Обновляет элементы управления GUI и обрабатывает ввод пользователя. 

int Update(
  float fDeltaTime
);

Параметры

fDeltaTime
Количество времени, прошедшее с последнего вызова метода Update в секундах.

Возвращаемое значение

Если один из элементов управления изменил свое состояние, метод Update вернет идентификатор элемента управления. Если был вызван метод Leave и все элементы управления закончили свою конечную анимацию, метод Update вернет -1. Otherwise it returns 0.

Примечания

Вызывайте метод Update в функции кадра, чтобы обновить GUI. Время, прошедшее с момента последнего вызова функции кадра можно получить вызовом функции системного слоя Timer_GetDelta.

Требования
