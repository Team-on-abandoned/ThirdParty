hgeParticleSystem::Stop

Останавливает систему частиц. 

void Stop(
  bool bKillParticles = false
);

Параметры

bKillParticles
Если равно true, все активные частицы удаляются. Если равно false или опущено, то активные частицы останутся жить до тех пор, пока они не исчезнут обычным образом.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib

Fire, FireAt, GetAge
