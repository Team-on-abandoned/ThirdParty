hgeGUIObject::MouseMove

Уведомляет элемент о том, что курсор мыши движется. 

virtual bool MouseMove(
  float x,
  float y
);

Параметры

x
Координата X курсора мыши относительно левого верхнего угла элемента.
y
Координата Y курсора мыши относительно левого верхнего угла элемента.

Возвращаемое значение

Если элемент изменил свое состояние и хочет уведомить об этом вызывающего, MouseMove должен вернуть true. В противном случае false.

Примечания

В классе hgeGUIObject есть стандартная реализация этого метода, так что можете не переопределять этот метод, если он вам не нужен.

Требования

Заголовок: hgegui.h
