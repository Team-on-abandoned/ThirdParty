hgeResourceManager::GetFont

Возвращает указатель на шрифт по имени. 

hgeFont *GetFont(
  const char *name
);

Параметры

name
Название шрифта.

Возвращаемое значение

В случае успеха, возвращается указатель на объект класса hgeFont. В противном случае 0.

Примечания

Чтобы использовать шрифты в hgeResourceManager, необходимо объявить их в скрипте ресурса.

Если шрифт еще не был загружен, он будет загружен. В противном случае вернется указатель на существующий шрифт.

Требования

Заголовок: hgeresource.h
Библиотека импорта: hgehelp.lib
