HGE::Resource_EnumFolders

Перечисляет дирекотрии по заданной маске. 

char *Resource_EnumFolders(
  const char *wildcard = 0
);

Параметры

wildcard
Абсолютная или относительная маска включая специальные символы "?" и "*" чтобы начать новое перечисление, или 0 для того, чтобы продолжить начатое перечисление.

Возвращаемое значение

Возвращает имя следющей найденной директории, или 0, если директорий больше нет.

Примечания

Resource_EnumFolders возвращает только имя директории без пути. Она так же пропускает файлы, для перечисления файлов используйте Resource_EnumFiles.

Для перечисления всех директорий в некотором месте, используйте в маске специальный символ "*". Например: Resource_EnumFolders("data\*") начнет перечисление всех директорий внутри директории "data" которая расположена в директории приложения.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
