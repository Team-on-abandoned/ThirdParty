Конструкторы кдасса hgeDistortionMesh

Чтобы создать и проинициализировать объект класса hgeDistortionMesh можно воспользоваться одним из следующих конструкторов: 

hgeDistortionMesh(
  int cols,
  int rows
);

hgeDistortionMesh(
  const hgeDistortionMesh &dm
);

Параметры

cols
Число столбцов сетки Минимальное значение - 2.
rows
Число строк сетки. Минимальное значение - 2.

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib
