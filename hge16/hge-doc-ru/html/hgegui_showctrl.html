hgeGUI::ShowCtrl

Показывает или скрывает элемент управления. 

void ShowCtrl(
  int id,
  bool bVisible
);

Параметры

id
Идентификатор элемента управления, который надо изменить.
bVisible
Если равно true элемент управления будет показан, в противном случае скрыт.

Требования

Заголовок: hgegui.h
Библиотека импорта: hgehelp.lib

Смотрите также

MoveCtrl, EnableCtrl
