HGE::Ini_GetString

Читает строку из файла инициализации. 

char *Ini_GetString(
  const char *section,
  const char *name,
  const char *def_val
);

Параметры

section
Имя секции файла инициализации.
name
Имя ключа.
def_val
Значение по умолчанию.

Возвращаемое значение

Возвращает указатель на временный буфер со строкой. Если требуемый ключ найден в текущем файле инициализации, его значение копируется в буфер. В противном случае копируется значение по умолчанию.

Примечания

Переменная системы HGE_INIFILE должна быть устрановлена перед вызовом этой функции. Имя секции регистро-независимое.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
