hgeParticleSystem::GetBoundingBox

Возвращает ограничивающий прямоугольник системы частиц. 

hgeRect *GetBoundingBox(
  hgeRect *rect
);

Параметры

rect
Указатель на объект hgeRect, куда требудется сохранить ограничивающий прямоугольник.

Возвращаемое значение

Возвращает значение в параметре rect.

Примечания

По умолчанию, отслеживание ограничивающего прямоугольника выключено, из соображений производительности. Если эта функциональность нужна, ее нужно включить явно посредствам вызова метода TrackBoundingBox.

Ограничивающий прямоугольник расчитывается на основании только положения частиц и размеры спрайтов частиц не учитываются.

Требования

Библиотека импорта: hgehelp.lib
