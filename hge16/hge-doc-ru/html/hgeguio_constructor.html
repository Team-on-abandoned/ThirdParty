Конструктор класса hgeGUIObject

Чтобы создать новые элементы управления GUI необходимо определить хотябы Конструктор и метод Render. Конструктор должен инициализировать следующие члены класса hgeGUIObject: id, bStatic, bVisible, bEnabled, rect. Смотрите Члены класса hgeGUIObject, для более подробной информации.

Используйте следующий синтаксис C++, чтобы определить класс производный от hgeGUIObject: 

class hgeCustomGUIControl : public hgeGUIObject
{
	hgeCustomGUIControl(...);
};

Требования

Заголовок: hgegui.h
