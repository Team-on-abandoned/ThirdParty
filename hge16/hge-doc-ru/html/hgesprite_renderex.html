hgeSprite::RenderEx

Рисует спрайт на экране с масштабированием и поворотом. 

void RenderEx(
  float x,
  float y,
  float rot,
  float hscale = 1.0f,
  float vscale = 0.0f
);

Параметры

x
Экранная координата x позиции вывода.
y
Экранная координата y позиции вывода.
rot
Угол поворота спрайта в радианах.
hscale
Горизонтальный коэффициент масштабирования. 1.0 означает исходный размер. Если параметр опущен, масштабирование не производится.
hscale
Вертикальный коэффициент масштабирования. 1.0 означает исходный размер. Если параметр равен 0.0 или опущен, hscale используется для обоих горизонтального и вертикального масштабирования.

Примечания

Масштабирование, поворот и позиционирование расчитываются относительно заданного центра спрайта. Смотрите более подробно в методе SetHotSpot.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib
