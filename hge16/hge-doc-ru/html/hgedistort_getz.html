hgeDistortionMesh::GetZ

Возвращает Z-глубину указанного узла сетки искажения. 

float GetZ(
  int col,
  int row
);

Параметры

col
Номер столбца узла сетки. Нумерация начинается с 0.
row
Номер строки узла сетки. Нумерация начинается с 0.

Возвращаемое значение

Возвращает Z-глубину указанного узла сетки.

Примечания

Если z-буфер не используется, значение z-глубины игнорируется.

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib

Смотрите также

SetZ, GetColor, GetDisplacement
