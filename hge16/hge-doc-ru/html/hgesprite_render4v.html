hgeSprite::Render4V

Рисует спрайт в произвольный четырехугольник на экране. 

void Render4V(
  float x0,
  float y0,
  float x1,
  float y1,
  float x2,
  float y2,
  float x3,
  float y3
);

Параметры

x0
Координата x первой вершины.
y0
Координата y первой вершины.
x1
Координата x второй вершины.
y1
Координата y второй вершины.
x2
Координата x третей вершины.
y2
Координата y третей вершины.
x3
Координата x четвертой вершины.
y3
Координата y четвертой вершины.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib
