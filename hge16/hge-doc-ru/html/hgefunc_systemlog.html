HGE::System_Log

Пишет форматированное сообщение в файл протокола. 

void System_Log(
  const char *format,
  ...
);

Параметры

format
форматная строка в стиле функуции printf стандартной библиотеки.
...
Необязательный список аргументов, который должен быть записан в файл протокола.

Примечания

Смотри справку по компилятору или руководство языка С для более подробной информации о форматной строке.

Чтобы данная функция работала, необходимо установить переменную системы HGE_LOGFILE до вызова функции System_Initiate.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
