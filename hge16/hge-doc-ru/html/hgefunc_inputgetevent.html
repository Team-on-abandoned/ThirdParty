HGE::Input_GetEvent

Берет следуюшее событие ввода из очереди. 

bool Input_GetEvent(
  hgeInputEvent *event
);

Параметры

event
Указатель на структуру hgeInputEvent, куда необходимо сохранить состояние.

Возвращаемое значение

Если событие было скопировано в указанную структуру, возвращается true. Если в очереди больше нет событий, возварщается false.

Примечания

Все полученные события очищаются в конце каждого кдра.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
