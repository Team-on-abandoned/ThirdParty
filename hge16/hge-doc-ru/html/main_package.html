Содержание пакета

HGE 1.53 Архив должен содержать следующие файлы: 

hge.dll		  DLL библиотеки HGE 1.53
bass.dll		  DLL библиотеки BASS 2.1
license.txt	  Лицензионное соглашение HGE
readme.txt	  Краткое описание HGE
doc\		  Он-лайн документация HGE
  html\		    Файлы документации
  index.html	    Указатель документации
include\		  Заголовки HGE
  hge.h		    Заголовок системного слоя
  hgeanim.h	    Заголовок класса hgeAnimation
  hgecolor.h	    Заголовок класса hgeColor
  hgedistort.h	    Заголовок класса hgeDistortionMesh
  hgefont.h	    Заголовок класса hgeFont
  hgegui.h	    Заголовоки классов hgeGUI и hgeGUIObject
  hgeguictrls.h	    Заголовок для стандартных элементов управления
  hgeparticle.h 	    Заголовок класса hgeParticleSystem
  hgerect.h	    Заголовок класса hgeRect
  hgeresource.h	    Заголовок класса hgeResourceManager
  hgesprite.h	    Заголовок класса hgeSprite
  hgevector.h	    Заголовок класса hgeVector
lib\		  Библиотеки импорта HGE
  bc\		    Библиотеки для Borland C++/C++ Builder
    hge.lib	      Библиотека системного слоя HGE
    hgehelp.lib	      Библиотека вспомогательных классов HGE
  gcc\		    Библиотеки для MinGW/Dev-C++
    hge.a		      Библиотека системного слоя HGE
    hgehelp.a	      Библиотека вспомогательных классов HGE
  vc\		    Библиотеки для VC++/Intel C++/Codewarrior
    hge.lib	      Библиотека системного слоя HGE
    hgehelp.lib	      Библиотека вспомогательных классов HGE
tools\		  Инструменты HGE
  particleed\	    Редактор систем частиц HGE
    hge.dll	      DLL библиотеки HGE 1.53
    particle1.psi	      Начальная установка #1
    particle2.psi	      Начальная установка #2
    particle3.psi	      Начальная установка #3
    particle4.psi	      Начальная установка #4
    particle5.psi	      Начальная установка #5
    particle6.psi	      Начальная установка #6
    particle7.psi	      Начальная установка #7
    particle8.psi	      Начальная установка #8
    particle9.psi	      Начальная установка #9
    particleed.exe	      Бинарий редактора систем частиц
    particleed.ini	      Инициализационный файл редактора систем частиц
    particleed.paq	      Ресурс файл редактора систем частиц
    particles.png	      Текстуры частиц
    readme.txt	      Краткое описание редактора систем частиц
tutorials\	  Коды примеров HGE
  precompiled\	    Скомпилированные примеры и медиа-файлы

    hge_tut08.exe	      Скомпилированный пример 08
    hge.dll	      DLL библиотеки HGE 1.53
    bass.dll	      DLL библиотеки BASS 2.1
    bg.png	      Тайл заднего фона 1
    bg2.png	      Тайл заднего фона 2
    cursor.png	      Изображение курсора мыши

    font1.png	      Растровое изображение шрифта 1
