hgeGUIText::printf

Печатает форматированную строку в текст элемента управления. 

void printf(
  const char *format,
  ...
);

Параметры

format
Строка в стиле функции printf языка C.
...
Необязательный список аргументов.

Примечания

Смотрите справку по компилятору или документацию по языку C для более подробной информации по спецификации формата строки.

Требования

Заголовок: hgeguictrls.h
Библиотека импорта: hgehelp.lib
