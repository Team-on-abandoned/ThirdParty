HGE::Ini_GetFloat

Читает значение с плавающей запятой из файла инициализации. 

float Ini_GetFloat(
  const char *section,
  const char *name,
  float def_val
);

Параметры

section
Имя секции в файле инициализации.
name
Имя ключа.
def_val
Значение по умолчанию.

Возвращаемое значение

Возварщает значение прочитанное из файла инициализации. Если файл, секция или ключ не были найдены, то возвращается значение по умолчанию.

Примечания

Переменная системы HGE_INIFILE должна быть устрановлена перед вызовом этой функции. Имя секции регистро-независимое.

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib
