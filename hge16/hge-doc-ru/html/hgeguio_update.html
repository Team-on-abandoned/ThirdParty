hgeGUIObject::Update

Вызывается каждый раз, когда GUI обновляется, чтобы сделать возможной анимацию элементов. 

virtual void Update(
  float fDeltaTime
);

Параметры

fDeltaTime
Количество времени, прошедшее с момента последнего вызова метода Update в секундах.

Примечания

В классе hgeGUIObject есть стандартная реализация этого метода, так что можете не переопределять этот метод, если он вам не нужен.

Требования

Заголовок: hgegui.h
