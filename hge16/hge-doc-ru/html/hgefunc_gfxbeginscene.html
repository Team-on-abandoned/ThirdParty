HGE::Gfx_BeginScene

Начинает вывод графики. 

bool Gfx_BeginScene(
  HTARGET target = 0
);

Параметры

target
Необязательный параметр, идинтификатор цели вывода, используемый для отображения графики. Если значение равно 0, или опущено, используется экран.

Возвращаемое значение

В случае успешного завершения, возвращается true. В противном случае false.

Примечания

Функция Gfx_BeginScene должна вызываться перед любыми вызовами функций рисования в HGE или перед методами рисования вспомогательных классов.

Если функция завершается аварийно, вы можете получить сообщение об ошибке вызовом функции System_GetErrorMessage. Если задана переменная системы HGE_LOGFILE, то сообщение об ошибке будет также записано в файл протокола.

Для завершения вывода графики, необходимо вызвать функцию Gfx_EndScene. Вложенные вызовы Gfx_BeginScene / Gfx_EndScene недопустимы.
