Настройка проекта HGE

Чтобы использовать движок HGE сначала необходимо указать компилятору, где брать требуемые файлы заголовков и импортные библиотеки. Все заголовочные файлы общие и находятся в директории "include". Библиотеки следует брать из соответствующей вашему компилятору директории:

  • Для Visual C++ и Intel C++ используйте библиотеки из директории "lib\vc".
  • Для Borland C++ и C++ Builder используйте библиотеки из директории "lib\bc".
  • Для MinGW32 используйте библиотеки из директории "lib\gcc".
  • Для Metrowerks Codewarrior использйте библиотеки либо Visual C++, либо Borland C++.

Примеры, как следует настраивать различные компиляторы и создавать пустой HGE проект:

Visual C++ 6.0
Visual Studio .NET
Borland C++ Builder 4
Dev C++ 5
