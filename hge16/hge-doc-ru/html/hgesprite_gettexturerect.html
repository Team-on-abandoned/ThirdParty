hgeSprite::GetTextureRect

Возвращает текущую область текстуры спрайта. 

void GetTextureRect(
  float *x,
  float *y,
  float *w,
  float *h
);

Параметры

x
Указатель на переменную типа float, куда необходимо сохранить координату X левого верхнего угла текущей области тексуры.
y
Указатель на переменную типа float, куда необходимо сохранить координату Y левого верхнего угла текущей области тексуры.
w
Указатель на переменную типа float, куда необходимо сохранить ширину текущей области тексуры.
h
Указатель на переменную типа float, куда необходимо сохранить ширину текущей области тексуры.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib

Смотрите также

SetTextureRect, SetTexture, GetTexture
