Конструкторы класса hgeParticleSystem

Чтобы создать и проинициализировать объект класса hgeParticleSystem можно воспользоваться следующими конструкторами: 

hgeParticleSystem(
  const char *filename,
  hgeSprite *sprite,
  float fps=0
);

hgeParticleSystem(
  hgeParticleSystemInfo *psi,
  float fps=0
);

hgeParticleSystem(
  const hgeParticleSystem &ps
);

Параметры

filename
Файл описания системы частиц. Этот файл должен содержать дисковый образ структуры hgeParticleSystemInfo, описывающей систему частиц.
sprite
Указатель на созданный объект класса hgeSprite, который будет использован в системе.
psi
Указатель на созданную структур hgeParticleSystemInfo, описывающую систему частиц.
fps
Частота обновления (количество раз в секунду) системы частиц. Используйте этот параметр, чтобы поддерживать одинаковое поведение системы частиц, независимо от частоты кадров и уменьшить немного вычисления. Если значение равно 0 или опущено, тогда система частиц будет обновляться каждый раз при вызове метода Update.

Примечания

Файл описания файловой системы загружается с помощью функции Resource_Load, поэтому все возможности этой функции доступны.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib
