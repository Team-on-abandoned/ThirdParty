Конструктор класса hgeResourceManager

Создать и проинициализирует объект класса hgeResourceManager можно используя следующий конструктор: 

hgeResourceManager(
  const char *scriptname = 0
);

Параметры

scriptname
Имя файла скрипт ресурса. Если равно 0 или опущено, то скрипт не используется.

Примечания

Хотя приходится создавать скприт ресурсов для управления сложными ресурсами, вы всегда можете пользловаться классом hgeResourceManager и без скрипта. Он позволит вам обращаться к простым ресурсам, текстурам, звуковым объектам и музыке по их названиям файлов.

Все ресурсы, загруженные объектом класса hgeResourceManager будут удалены из памяти в деструкторе hgeResourceManager.

Скрипт ресурсов загружается функцией Resource_Load, поэтому все ее особенности применимы к этому ресурсу.

Требования

Заголовок: hgeresource.h
Библиотека импорта: hgehelp.lib
