hgeGUIObject::Focus

Вызывается когда элемент управления получает или теряет фокус ввода клавиатуры. 

virtual void Focus(
  bool bFocused
);

Параметры

bFocused
Если элемент получает фокус, bFocused равно true. Если элемнт теряет фокус, bFocused равно false.

Примечания

В классе hgeGUIObject есть стандартная реализация этого метода, так что можете не переопределять этот метод, если он вам не нужен.

Требования

Заголовок: hgegui.h
