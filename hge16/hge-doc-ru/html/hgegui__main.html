Вспомогательный класс hgeGUI

hgeGUI является вспомогательным классом HGE для создания и управления пользовательским интерфейсом.

Методы класса hgeGUI

Конструктор Создает и инициализирует объект класса hgeGUI.
  
AddCtrl Добавляет элемент управления в GUI.
DelCtrl Удаляет элемент управления из GUI.
GetCtrl Возвращает указатель на элемент управления по его идентификатору.
MoveCtrl Перемещает элемент управления.
ShowCtrl Показывает или скрывает элемент управления.
EnableCtrl Делает доступным или недоступным элемент управления.
  
SetNavMode Устанавливает режим навигации в GUI.
SetCursor Устанавливает спрайт курсора мыши.
SetFocus Устанавливает фокус ввода на элемент управления.
GetFocus Возвращает текущий фокус ввода.
Enter Запускает начальную анимацию GUI.
Leave Запускает завершающую анимацию GUI.
  
Update Обновляет GUI и обрабатывает ввод пользователя.
Render Рисует GUI на экране.

Дополнительная информация

hgeGUIObject Абстрактынй базовый класс для произвольных элементов управления GUI.
  
hgeGUIText Класс текстового элемента GUI.
hgeGUIButton Класс кнопки GUI.
hgeGUISlider Класс бегунка GUI.
  
Macros Макросы для простейшего управления элементами управления.

Требования

Заголовок: hgegui.h
