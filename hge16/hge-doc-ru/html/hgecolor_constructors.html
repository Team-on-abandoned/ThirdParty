Конструктор класса hgeColor

Чтобы создать и проинициализировать объекты класса hgeColor можете воспользоваться одним из следующих конструкторов: 

hgeColor();

hgeColor(
  DWORD color
);

hgeColor(
  float r,
  float g,
  float b,
  float a
);

hgeColor(
  const hgeColor &col
);

Параметры

color
Аппаратное значение цвета для инициализации hgeColor.
r
Значение красной составляющей в пределах от 0.0 до 1.0.
g
Значение зеленой составляющей в пределах от 0.0 до 1.0.
b
Значение синей составляющей в пределах от 0.0 до 1.0.
a
Алфа значение в пределах от 0.0 до 1.0.

Требования

Заголовок: hgecolor.h
