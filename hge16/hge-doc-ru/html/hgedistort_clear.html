hgeDistortionMesh::Clear

Удаляет перемещения и раскраску сетки. 

void Clear(
  DWORD color = 0xFFFFFFFF,
  float z = 0.5f
);

Параметры

color
Цвет, который должен быть установлен всем узлам сетки. Если параметр опущен, берется значение 0xFFFFFFFF.
z
Z-глубина, которая должна быть назначена всем узналм сетки. Если парамтр опущен, берется значение 0.5f.

Требования

Заголовок: hgedistort.h
Библиотека импорта: hgehelp.lib
