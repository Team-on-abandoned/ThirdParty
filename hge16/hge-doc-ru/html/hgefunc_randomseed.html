HGE::Random_Seed

Устанавливает начальное значение генератора случайных чисел. 

void Random_Seed(
  int seed = 0
);

Параметры

seed
Начальное значение генератора случайных чисел. Если равно 0, в качестве значения выбирается текущее время.

Примечания

Генератор случайных чисел выдает псевдо-случайные последовательности. Начальное значение определяет следющую сгенерированную последовательность. Поэтому вы можете извлечь пользу из этой функции, определяя длинные серии чисел с одинаковым начальным числом (например, это может быть полезно в генерации карт/уровней).

Чтобы поставить случайную последовательность в случайную точку, используйте 0 в качестве начального значения.
