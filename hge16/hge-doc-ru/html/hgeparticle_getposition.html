hgeParticleSystem::GetPosition

Возвращает положение системы частиц. 

void GetPosition(
  float *x,
  float *y
);

Параметры

x
Указатель на переменную типа float, куда необходимо сохранить X-координату системы частиц.
y
Указатель на переменную типа float, куда необходимо сохранить Y-координату системы частиц.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib

Смотрите также

MoveTo, Transpose, GetTransposition
