Члены класса hgeRect

У класса hgeRect есть public члены класса, которые можно изменять напрямую: 

float x1;
float y1;
float x2;
float y2;

Члены класса

x1
Координата X левого верхнего угла.
y1
Координата Y левого верхнего угла.
x2
Координата X правого нижнего угла.
y2
Координата Y правого нижнего угла.

Требования

Заголовок: hgerect.h
