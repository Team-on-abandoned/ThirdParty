hgeGUIObject::KeyClick

Уведомляет элемент управления о том, что нажата клавиша. 

virtual bool KeyClick(
  int key,
  int chr
);

Параметры

key
Виртуальный код нажатой клавиши. Смотриет Виртуальные коды клавиш для полного списка.
chr
Код символа, учитывая состояние клавиатуры и локаль.

Возвращаемое значение

Если элемент изменил свое состояние и хочет чтобы вызывавший был уведомлен, метод KeyClick должен вернуть true. В противном случае должен вернуть false.

Примечания

В классе hgeGUIObject есть стандартная реализация этого метода, так что можете не переопределять этот метод, если он вам не нужен.

Требования

Заголовок: hgegui.h
