HGE Vertex structure

Структура hgeVertex описывает вершины, из которых состоят графические примитивы HGE. 

struct hgeVertex
{
  float  x, y;
  float  z;
  DWORD  col;
  float  tx, ty;
};

Члены

x
Экранная координата X.
y
Экранная координата Y.
z
Z-глубина вершин принимает значение от 0.0 до 1.0.
Если Z-глубина не используется, то это значение игнорируется.
col
Цвет вершины и прозрачность.
tx
Текстурная координата X принимает значение от 0.0 до 1.0.
ty
Текстурная координата Y принимает значение от 0.0 до 1.0.

Примечания

Если вы указываете значиние текстурной координаты выходящее за пределы 0.0-1.0, данные текстуры будут повсторены через каждое целое значение длины текстуры. Например, очень легко реализовать эффект "rotozoom".

Требования

Заголовок: hge.h
