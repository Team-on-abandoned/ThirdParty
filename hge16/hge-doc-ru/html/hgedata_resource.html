Идентификаторы ресурсов в HGE

Следующие типы идентификаторов ресурсов используются для учета внутренних ресурсов HGE: 

typedef DWORD  HTEXTURE;
typedef DWORD  HTARGET;
typedef DWORD  HEFFECT;
typedef DWORD  HMUSIC;
typedef DWORD  HSTREAM;
typedef DWORD  HCHANNEL;
HTEXTURE
Идентификатор текстуры
HTARGET
Идентификатор цели рендеринга
HEFFECT
Идентификатор звукового эффекта
HMUSIC
Идентификатор музыки
HSTREAM
Идентификатор сжатого звукового потока
HCHANNEL
Идентификатор активного звукового канала

Требования

Заголовок: hge.h
