Конструкторы класса hgeVector

Чтобы создать и проинициализировать объекты класса hgeVector можете воспользоваться одним из следующих конструкторов: 

hgeVector();

hgeVector(
  float x,
  float y
);

hgeVector(
  const hgeVector &vec
);

Параметры

x
Компонента X.
y
Компонента Y.

Требования

Заголовок: hgevector.h
