hgeSprite::GetBoundingBoxEx

Возвращает масштабированный и повернуты ограничивающий прямоугольник. 

hgeRect *GetBoundingBoxEx(
  float x,
  float y,
  float rot,
  float hscale,
  float vscale,
  hgeRect *rect
);

Параметры

x
Экранная координата x спрайта.
y
Экранная координата y спрайта.
rot
Угол поворота спрайта в радианах.
hscale
Горизонтальный коэффициент масштабирования. 1.0 означает исходный размер.
vscale
Вертикальный коэффициент масштабирования. 1.0 означает исходный размер.
rect
Указатель на объект hgeRect, где будет сохранен ограничивающий прямоугольник.

Возвращаемое значение

Возвращает параметр rect.

Требования

Заголовок: hgesprite.h
Библиотека импорта: hgehelp.lib
