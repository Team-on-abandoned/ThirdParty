hgeVector::Angle

Возвращает угол между двумя векторами или между вектором и осью X. 

float Angle(
  hgeVector *vector = 0
);

Параметры

vector
Необязательный параметр, указатель на вектор, с которым нужно вычислить угол. Если равен 0 или опущен, то угол вычилсяется между осью X.

Возвращаемое значение

Возвращает угол между двумя векторами или между вектором и осью X в радианах.

Требования

Заголовок: hgevector.h
Библиотека импорта: hgehelp.lib
