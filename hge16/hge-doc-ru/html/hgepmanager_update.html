hgeParticleManager::Update

Обновляет все активные системы частиц. 

void Update(
  float fDeltaTime
);

Параметры

fDeltaTime
Время, прошедшее с предидущего вызова метода Update в секундах.

Примечания

Вызывайте метод Update в функции кадра для обновления всех систем частиц, принадлежащих менеджеру. Время, прошедшее после последнего вызова функции кадра можно получить функцией системного слоя Timer_GetDelta.

Требования

Заголовок: hgeparticle.h
Библиотека импорта: hgehelp.lib
