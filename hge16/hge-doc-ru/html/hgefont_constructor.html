Конструкторы класса hgeFont

Чтобы создать и проинициализировать объект класса hgeFont можно воспользоваться одним из следующих конструкторов: 

hgeFont(
  const char *filename
);

hgeFont(
  const hgeFont &font
);

Параметры

filename
Файл описания шрифта.

Примечания

Файл описания шрифта загружается функцией Resource_Load, поэтому вся функциональность для работы с ресурсами доступна.

Требования

Заголовок: hgefont.h
Библиотека импорта: hgehelp.lib
