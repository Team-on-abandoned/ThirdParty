hgeGUI::SetCursor

Устанавливает спрайт курсора мыши. 

void SetCursor(
  hgeSprite *sprite
);

Параметры

sprite
Указатель на спрайт, который будет использован в качестве курсора мыши. Если равно 0, курсор не будет отображаться.

Примечания

По умолчанию, GUI не отображает курсор мыши, поэтому нужно установить хотябы один раз или рисовать его самому. Вы можете изменить курсор в любое время.

В отличие от элементов управления, GUI не управляет курсорами сама, поэтому необходимо сохранять указатели и освобождать курсоры, когда они больше не нужны самостоятельно.

Требования
