HGE::Resource_AttachPack

Присоединяет архив ресурсов. 

bool Resource_AttachPack(
  const char *filename,
  const char *password = 0
);

Параметры

filename
Имя файла архива ресурсов (отосительно директории приложения или абсолютный путь).
password
Необязательный пароль.

Возвращаемое значение

В случае успеха, возвращает true. В противном случае false.

Примечания

Если функция завершается аварийно, вы можете получить сообщение об ошибке при помощи вызова System_GetErrorMessage. Так же если установлена перменная среды HGE_LOGFILE, то сообщение об ошибке запишется в файл протокола.

Во время загрузки ресурсов, архивы присоединенные последними имеют больший приоритет. Смотрите секцию Compiling resource pack для более подробной информации об архивах ресурсов.

Когда архив ресурсов больше не нужне, необходимо вызвать функцию Resource_RemovePack. Все неотсоединенные архивы автоматически отсоединяются во время вызова Release.
