HGE::Random_Int

Генерирует случайное число типа int. 

int Random_Int(
  int min,
  int max
);

Параметры

min
Нижняя граница.
max
Верхняя граница.

Возвращаемое значение

Возвращает слчаучаное число между min и max (включительно).

Требования

Заголовок: hge.h
Библиотека импорта: hge.lib

Смотрите также

Random_Float, Random_Seed
