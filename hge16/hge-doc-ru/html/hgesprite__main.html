Вспомогательный класс hgeSprite

hgeSprite является вспомогятельным классом HGE для реализации спрайтов.

Методы hgeSprite

Конструкторы Создает и инициализируют объекты класса hgeSprite.
Операторы Операторы hgeSprite.
  
Render Рисует спрайт на экране.
RenderEx Рисует спрайт с масштабированием и поворотом.
RenderStretch Рисует наклоненный спрайт.
Render4V Рисует спрайт в произвольный цетырехугольник на экране.
  
SetTexture Устанавливает текстуру для спрайта.
SetTextureRect Устанавливает область текстуры для спрайта.
SetColor Устанавливает раскраску и альфу в заданных вершинах или для всего спрайта.
SetZ Устанавливает Z-глубину для заданных вершин или всего спрайта.
SetBlendMode Устанавливает режим смешивания для спрайта.
SetHotSpot Устанавливает центр спрайта.
SetFlip Отражает спрайт по горизонтали и/или вертикали.
  
GetTexture Возвращает текущую текстуру спрайта.
GetTextureRect Возвращает текущую область текстуры спрайта.
GetColor Возвращает цвет указанной вершины спрайта.
GetZ Возвращает Z-глубину указанной вершины спрайта.
GetBlendMode Возвращает установленный режим смешивания спрайта.
GetHotSpot Возвращает центр спрайта.
GetFlip Возвращает текущее отражение.
  
GetWidth Возвращает ширину спрайта.
GetHeight Возвращает высоту спрайта.
GetBoundingBox Возвращает ограничивающий прямоугольник спрайта.
GetBoundingBoxEx Возвращает отмасштабированный и повернуты ограничивающий прямоугольник спрайта.
